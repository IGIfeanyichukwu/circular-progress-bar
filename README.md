# Circular Progress Bar

#### _...displays circular progress bars_

Cicular Progress Bar displays progress in a circular progress bar as value are typed in the input box. It is the idea behind [Cyprobar](http://github.com/IGIfeanyichukwu/cyprobar), a javascript libary for creating circular progress bars.
